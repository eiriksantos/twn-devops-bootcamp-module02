# Basic Linux Commands

### Cheatsheet

Print working directory
```
$ pwd
```

Display folders and files in the current directory
```
$ ls
```

Changing directory
```
$ cd
$ cd Documents
```

Create directory
```
$ mkdir
$ mkdir test-project
```

Create a file
```
$ touch
$ touch main.py
```

Remove a file
```
$ rm
$ rm main.py
```

Remove a directory
```
$ rm -r test-project
```

Rename a file
```
$ mv web-application java-app
```

Copy a file/folder
```
$ cp Readme.md Readme-test.md
$ cp -r java-app my-project
```

Display the contents of a file
```
$ cat Readme.md
```

Display info of your machine
```
$ uname -a
$ cat /etc/os-release
$ lscpu
$ lsmem
```

Execute commands as superuser
```
$ sudo
```
