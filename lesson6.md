# Basic Linux Commands

### Cheatsheet

Print working directory
```
$ pwd
```

Display folders and files in the current directory
```
$ ls
```

Changing directory
```
$ cd
$ cd Documents
```

Create directory
```
$ mkdir
$ mkdir test-project
```

Create a file
```
$ touch
$ touch main.py
```

Remove a file
```
$ rm
$ rm main.py
```

Remove a directory
```
$ rm -r test-project
```
